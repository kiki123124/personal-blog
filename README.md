# My Space - Personal Blog & Music Share

A modern, minimalist React application built with Next.js, featuring a personal blog, music sharing, and an admin dashboard.

## Features

- **Personal Homepage**: Elegant introduction with smooth animations.
- **Blog**: Markdown-based blog system with list and detail views.
- **Music**: Audio player for sharing and listening to tracks.
- **Admin Dashboard**: Secure interface to write posts and upload music files.
- **Tech Stack**: Next.js 15 (App Router), Tailwind CSS, Framer Motion, TypeScript.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open the app**:
    Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: App router pages and API routes.
- `src/content/posts`: Markdown files for blog posts.
- `src/lib`: Helper functions.
- `public/music`: Uploaded music files.
